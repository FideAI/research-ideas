# Security and Sensitive Information

This repository is public. Do not open issues or pull requests containing:

- credentials, API keys, tokens, or secrets;
- private participant data;
- identifying pastoral, counseling, student, family, or ministry details;
- confidential partner information;
- sensitive reviewer notes;
- unpublished raw model outputs from restricted evaluations.

If you find a security issue or accidentally disclose sensitive information,
contact Fide AI privately through the contact path at `https://fideai.org/participate`
or by emailing `alex@fideai.org`.

For ordinary research questions, use GitHub issues.
